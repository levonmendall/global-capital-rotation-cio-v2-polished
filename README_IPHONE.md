# Create hidden files manually from iPhone

Create these exact GitHub paths:

1. `.github/workflows/validate.yml`
   Paste `01_VALIDATE_WORKFLOW.yml`

2. `.github/workflows/daily_refresh.yml`
   Paste `02_DAILY_REFRESH_WORKFLOW.yml`

3. `.streamlit/config.toml`
   Paste `03_STREAMLIT_CONFIG.toml`

4. `.streamlit/secrets.example.toml`
   Paste `04_STREAMLIT_SECRETS_EXAMPLE.toml`

Never paste real credentials into the example secrets file.
