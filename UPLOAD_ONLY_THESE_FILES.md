# iPhone replacement instructions

Upload these files to the root of the V2 GitHub repository and replace any existing versions:

1. `app.py`
2. `engine.py`
3. `report_builder.py`
4. `requirements.txt`

Commit message:

`Deploy polished V2 command center`

After the green GitHub validation check appears:

1. Open the V2 Streamlit app.
2. Tap **Manage app**.
3. Tap the three-dot menu and select **Reboot app**.
4. Reload Safari.

The first screen must say **Chief Investment Officer Command Center** and show sidebar navigation for Command Center, My Portfolio, Capital Rotation, Risk & Regime, CIO Committee, Reports, and System Health.
